Usage: mdl2map [option] <your mdl file> <name of your wanted map>
Options:
	-h : help page
	-s [float] : scale of mdl (default 0.025)
        -g [float] : grid snap (default 0.00)
	-x [float] : translation on X axis (default 0.0)
	-y [float] : translation on Y axis (default 0.0)
	-z [float] : translation on Z axis (default 0.0)
